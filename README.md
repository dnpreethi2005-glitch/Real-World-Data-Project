# 🛒 Real World Data Project - Retail Sales Analysis and Prediction

## 📌 Overview
This project focuses on analyzing retail sales data and predicting total sales using Machine Learning. The project includes data exploration, visualization, and a Linear Regression model for prediction.

---

## 📂 Dataset
- Retail Sales Dataset
- Contains customer information, product categories, quantity purchased, price per unit, and total amount.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

---

## 📁 Project Structure

```
Real_World_Data_Project

├── retail_sales_dataset.csv

└── README.md
```

## 📊 Features

### 🔹 Data Analysis
- Display dataset information
- Check missing values
- Generate statistical summaries

### 🔹 Data Visualization
- Sales by Product Category
- Age Distribution
- Quantity Distribution
- Gender Distribution

### 🔹 Machine Learning
- Linear Regression Model
- Train-Test Split
